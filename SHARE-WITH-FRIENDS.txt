SonicScribe - Sharing Guide
===========================

YES, you can share SonicScribe with a friend. Give them the entire folder:

  dist\SonicScribe\

They must keep ALL files in that folder together (do not copy only the .exe).

WHAT YOUR FRIEND NEEDS ON THEIR PC
----------------------------------
1. Windows 10 or 11 (64-bit)
2. Python 3.12 (or 3.11) installed from python.org
3. These Python packages (run in Command Prompt or PowerShell):

   pip install faster-whisper torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu124

   (Use the CPU-only PyTorch index if they do not have an NVIDIA GPU.)

4. FFmpeg (optional but recommended for video files):
   winget install Gyan.FFmpeg

HOW TO SHARE
------------
- Zip the whole "SonicScribe" folder and send via Google Drive, OneDrive, USB stick, etc.
- Or upload to a file-sharing service (~200 MB compressed).

FIRST RUN ON THEIR PC
---------------------
1. Unzip the folder anywhere (e.g. Desktop\SonicScribe)
2. Double-click SonicScribe.exe
3. Open Settings and click "Auto-detect" for Python
4. Re-detect GPU if they have an NVIDIA card
5. Start transcribing

NOTES
-----
- The app runs 100% locally. No account or internet needed after setup.
- Transcripts save to Documents\SonicScribe\Outputs
- Settings save to %LocalAppData%\SonicScribe
